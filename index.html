About Me

Hi, my name is Shane Short and I'm a data engineer originally from Kildare, Ireland currently based in Perth, WA. My focus professionally has been in data warehouse development, ETL, and test automation, however I also enjoy exploring many areas of software engineering in my own time. A far from exhaustive list being lightweight game development, cryptography, machine learning, and parallel computing for numerical analysis.
I love to start projects and never get back to them, so hopefully this will be a solution to this problem, rather than another instance of it. In the Projects section of this site, you'll find a list of projects I've started at some point or another. You'll also see tags on each of these projects showing roughly how 'done' they really are. With a bit of work, these should start to trend towards 100% over time.
